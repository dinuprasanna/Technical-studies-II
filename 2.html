This simple Arduino project uses a potentiometer to control a 3-driver LED driver. Thus it gives control over the colour of the LED. The Story It has been a long time since I used Arduino; surprisingly, it has also been a long time since I have been to a party. My objective when I started making this was to test if my board works using the built-in LED. But instinctively, I started playing with the driver and gradually created these potentiometer-controlled RGB lights. The evolution of this is fascinating as our desires and needs get translated into what we make unconsciously. This was the first thing that I ended up making in this course. Not only did I enjoy making this, but it also gave me the confidence to move forward and create cool stuff with Arduino.
